# 5. Longest Palindromic Substring

## Difficulty

Easy **`Medium`** Hard

## Description

Given a string `s`, return *the longest palindromic substring* in `s`.

### Example 1:

<pre>
<b>Input:</b> s = "babad"
<b>Output:</b> "bab"
<b>Note:</b> "aba" is also a valid answer.
</pre>

### Example 2:

<pre>
<b>Input:</b> s = "cbbd"
<b>Output:</b> "bb"
</pre>

### Example 3:

<pre>
<b>Input:</b> s = "a"
<b>Output:</b> "a"
</pre>

### Example 4:

<pre>
<b>Input:</b> s = "ac"
<b>Output:</b> "a"
</pre>

### Constraints:

* `1 <= s.length <= 1000`
* `s` consist of only digits and English letters (lower-case and/or upper-case),
